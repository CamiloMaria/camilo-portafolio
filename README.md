# Camilo Maria Portfolio

Personal portfolio website with a retro arcade neon aesthetic. Built to showcase frontend skills to recruiters.

## Features

- 🎮 Retro arcade theme with neon aesthetics
- ✨ Floating particles background
- 📄 Resume modal with PDF preview
- 📱 Fully responsive
- ♿ Accessibility support (prefers-reduced-motion)

## Tech Stack

- **Next.js 16** - React framework
- **React 19** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS v4** - Styling
- **Framer Motion** - Animations

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Customization

- Edit personal info in `lib/portfolio-data.ts`
- Modify colors in `app/globals.css`
- Adjust particles in `components/ui/particles.tsx`

## Deploy

Deploy to Vercel:

```bash
npm run build
vercel deploy
```
